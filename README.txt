Temperature Conversion App
A simple web application that converts temperatures between Celsius and Fahrenheit.

Features

Convert temperatures from Celsius to Fahrenheit
Convert temperatures from Fahrenheit to Celsius
User-friendly interface with input field and checkboxes for selecting conversion type
Displays result in a paragraph element
How it Works

Enter a temperature value in the input field.
Select the conversion type by checking the corresponding checkbox (Celsius to Fahrenheit or Fahrenheit to Celsius).
Click the "Convert" button to perform the conversion.
The result will be displayed in the paragraph element below the button.
Code Structure

HTML: index.html contains the user interface and links to the CSS and JavaScript files.
CSS: style.css styles the user interface.
JavaScript: script.js contains the conversion logic and event handling.
Installation

Clone the repository to your local machine.
Open index.html in a web browser to run the application.
